This repository contains the source code for my **personal portfolio website**, built to showcase my projects, skills, and professional journey. The portfolio is designed with a clean UI, smooth interactions, and responsiveness across devices.

-------------------------------

Features

- Modern, responsive design
- Interactive UI with smooth animations
- Dedicated sections for:
  - About Me
  - Experience
  - Projects
  - Skills
  - Contact (EmailJS integration)
- Fully responsive for desktop and mobile
- Clean and maintainable codebase

---------------------------------
Tech Stack

- **Frontend:** React, JavaScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Email Service:** EmailJS
- **Build Tool:** Vite
